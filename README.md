# Sentinel-Android

🛡️ **Sentinel** 是一款基于 Root（Magisk）权限的 Android 进程守护与后台管理工具。

---

## 项目简介

这是一个由 AI 生成的 Android 项目，旨在提供一个基于 Root 权限的进程管理和后台控制工具。项目核心功能包括：

- 基于 `/proc` 文件系统的实时进程扫描
- Root 权限执行封装
- 应用级后台控制（安全策略）
- 系统白名单保护机制

> ⚠️ 本项目仅供学习、研究与高级用户自用，所有代码均由 AI 生成。

---

## 核心特性

### 1️⃣ 真实进程扫描
- 基于 `/proc` 文件系统扫描
- 获取 PID/UID、真实 cmdline 和对应应用包名

### 2️⃣ 应用级后台控制
- 首选 `am force-stop`（等价系统"强制停止"）
- 支持 UID/PID 级兜底终止
- 内置系统白名单，避免误杀关键服务

### 3️⃣ Root 执行抽象层
- 统一封装 `su` 调用
- 兼容 Magisk 授权模型
- 可控、可审计、易扩展

### 4️⃣ 场景化触发机制
- 支持锁屏、游戏模式等场景自动执行
- 可配置的白名单管理
